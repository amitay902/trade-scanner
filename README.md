# סורק תעלות 📈 — איתותים יומיים למסחר בטווח קצר

אפליקציית PWA שסורקת כל יום מסחר את כל המניות האמריקאיות בשווי שוק של 2 מיליארד דולר ומעלה, מאתרת מניות שנמצאות **בתחתית של תעלה עולה** (פולבק בתוך מגמת עלייה), ומציעה נקודת כניסה, יעד רווח, סטופ־לוס ותאריך דוחות — בדיוק לפי האסטרטגיה: RSI נמוך, מחיר מתחת ל-SMA20/50 אבל מעל SMA200, בתוך תעלה עולה שלא נשברה.

## מה מקבלים

- 📡 **איתותים יומיים** — סריקה אוטומטית לפני פתיחת וול סטריט (GitHub Actions, חינם)
- 🔔 **התראות Push לטלפון** — דרך אפליקציית ntfy החינמית: מניות חדשות, התקרבות ליעד/סטופ, דוחות קרובים
- 💼 **מעקב פוזיציות** — סימון "נכנסתי", עריכת יעדים, רישום מכירה עם סיבת יציאה
- 📊 **סטטיסטיקות** — אחוזי הצלחה, תוחלת לעסקה, Profit Factor, עקומת תשואה, פילוחים לפי חודש וסיבת יציאה
- 📱 **PWA** — מותקן כמו אפליקציה בטלפון ובמחשב, עובד גם אופליין

## התקנה — 10 דקות, הכול בחינם

### שלב 1 — יצירת ה-repo

1. צור חשבון (חינמי) ב-[github.com](https://github.com) אם אין לך.
2. צור repo חדש **ציבורי** (Public), למשל בשם `trade-scanner`.
3. העלה את כל הקבצים מהתיקייה הזו ל-repo (אפשר בגרירה: Add file → Upload files → לגרור את כל התוכן **כולל** התיקיות `scanner`, `data`, `icons`, `.github`).
   - אם ההעלאה בדפדפן לא כוללת את `.github/workflows/daily-scan.yml`, צור אותו ידנית: Add file → Create new file → הדבק את שם הנתיב המלא ואת התוכן.

### שלב 2 — הפעלת האתר (GitHub Pages)

1. ב-repo: **Settings → Pages**.
2. תחת Source בחר **Deploy from a branch**, ענף `main`, תיקייה `/ (root)` → Save.
3. אחרי ~2 דקות האפליקציה תהיה זמינה ב:
   `https://<המשתמש-שלך>.github.io/trade-scanner/`

### שלב 3 — הפעלת הסריקה היומית

1. ב-repo: לשונית **Actions** → אשר הפעלת workflows אם מתבקש.
2. בחר **Daily channel-up scan** → **Run workflow** להרצה ראשונה ידנית.
3. מעכשיו הסריקה תרוץ אוטומטית בכל יום מסחר בסביבות 15:45 שעון ישראל (לפני פתיחת וול סטריט).

### שלב 4 — התראות לטלפון (ntfy)

1. התקן את אפליקציית **ntfy** ([Android](https://play.google.com/store/apps/details?id=io.heckel.ntfy) / [iOS](https://apps.apple.com/us/app/ntfy/id1625396347)).
2. בתוך ntfy: **+ Subscribe to topic** והזן את הנושא שמופיע באפליקציה תחת ⚙️ הגדרות (ברירת המחדל מוטמעת שם).
3. מומלץ: החלף לנושא פרטי משלך — בחר שם אקראי וקשה לניחוש, עדכן אותו בהגדרות האפליקציה, וב-repo הוסף Secret בשם `NTFY_TOPIC` עם אותו ערך (**Settings → Secrets and variables → Actions → New repository secret**).

### שלב 5 — התקנת האפליקציה בטלפון

- **iPhone**: פתח את הכתובת ב-Safari → כפתור שיתוף → **Add to Home Screen**.
- **Android**: פתח ב-Chrome → תפריט ⋮ → **Install app**.

### שלב 6 (אופציונלי, מומלץ) — סנכרון פוזיציות לענן

בלי זה הפוזיציות נשמרות רק במכשיר, וההתראות על יעד/סטופ מחושבות רק כשפותחים את האפליקציה. עם זה — הסריקה היומית מכירה את הפוזיציות שלך ושולחת Push אמיתי:

1. ב-GitHub: **Settings (של החשבון) → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token**.
2. תן גישה רק ל-repo הזה, והרשאת **Contents: Read and write**.
3. באפליקציה: ⚙️ הגדרות → סנכרון לענן → הזן משתמש, שם repo וה-token → "סנכרן עכשיו".

> ⚠️ ה-token נשמר מקומית במכשיר שלך בלבד. אל תשתף אותו.

## איך הסריקה עובדת

| שלב | פירוט |
|---|---|
| יקום | כל המניות בארה"ב עם שווי שוק ≥ $2B (NASDAQ screener API; גיבוי: S&P 500+400) |
| נתונים | נרות יומיים ~10 חודשים מ-Yahoo Finance |
| תעלה עולה | רגרסיה לינארית על 45–110 ימי מסחר; שיפוע חיובי (רווח ≥8% בחלון), R² ≥ 0.55 |
| נקודת כניסה | מחיר בשליש התחתון של התעלה; RSI(14) ≤ 47; מתחת ל-SMA20; מעל SMA200; רצפת התעלה לא נשברה |
| איתות מחמיר | בנוסף: RSI ≤ 45 וגם מתחת ל-SMA50 |
| יעד רווח | ראש התעלה היום; אם יש שיא 52ש׳ ישן (מלפני התעלה) קרוב יותר — מעט מתחתיו |
| סטופ | ‏1.2×ATR יומי, או 1.7×ATR במניות תנודתיות (ATR>5%); תמיד מתחת לרצפת התעלה |
| העשרה | תאריך דוחות + יעד אנליסטים ממוצע לכל איתות |

## שאלות נפוצות

**אפשר לשנות את שעת הסריקה?** כן — ערוך את שורת ה-`cron` ב-`daily-scan.yml` (הזמן שם ב-UTC; ישראל בקיץ = UTC+3).

**אפשר לכוונן את האסטרטגיה?** כן — כל הפרמטרים מרוכזים בראש `scanner/scan.py` (RSI מקסימלי, R² מינימלי, מכפילי ATR ועוד).

**ה-repo חייב להיות ציבורי?** בשביל GitHub Pages בחשבון חינמי — כן. הקוד והאיתותים גלויים, אבל הפוזיציות שלך מסונכרנות רק אם בחרת בכך (וגם אז — שקול repo פרטי + שירות אירוח אחר אם זה רגיש לך).

**זה ייעוץ השקעות?** לא. זה כלי עזר טכני בלבד. כל החלטת מסחר היא באחריותך.
